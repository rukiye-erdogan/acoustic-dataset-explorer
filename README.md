# Acoustic Dataset Explorer

A lightweight open-source data engineering project for validating, standardising and analysing acoustic metadata.

## Features

- Acoustic metadata validation
- Dataset statistics
- Metadata quality checks
- Reproducible project structure
- Python-based processing
- Synthetic example metadata
